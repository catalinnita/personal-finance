# ActionButton

The `ActionButton` component.

## Usage

```tsx
import { ActionButton } from '...'

<ActionButton
  onClick={() => {}}
  disabled={true}
  className="value"
>
  {/* children */}
</ActionButton>
```

| Prop | Type | Default | Required | Description |

| `onClick` | `() => void` | — | Yes | — |
| `disabled` | `boolean` | — | Yes | — |
| `className` | `string` | — | Yes | — |
| `children` | `ReactNode` | — | Yes | — |
