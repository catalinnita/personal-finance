# SelectField

The `SelectField` component.

## Usage

```tsx
import { SelectField } from '...'

<SelectField
  value={/* string | number */}
  onChange={/* ChangeEventHandler<HTMLSelectElement> */}
  className="value"
>
  {/* children */}
</SelectField>
```
